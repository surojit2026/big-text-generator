# Big Text Generator - Create Large Text Online

A free online tool to instantly create and display **big, giant-sized text** with customizable font sizes (20px to 200px). Perfect for presentations, social media posts, headings, digital art, banners, and eye-catching displays!

**Live Demo:** [https://big-text.netstudy.in/](https://big-text.netstudy.in/)

## Features

- Adjustable font size from **20px to 200px** using a smooth slider
- Real-time preview – see changes instantly as you type or resize
- **Full Screen Mode** – display your big text across the entire screen (great for presentations or group viewing)
- One-click **Copy Text** button to copy the generated text to clipboard
- **Dark & Light Mode** toggle for comfortable viewing
- **Clear All** button to reset everything quickly
- Fully responsive and mobile-friendly – works perfectly on phones, tablets, and desktops
- No signup, no installation – 100% browser-based and free
- Ad-supported but clean and fast-loading interface

## How to Use

1. Visit **[https://big-text.netstudy.in/](https://big-text.netstudy.in/)**
2. Type or paste your text in the input box
3. Use the slider to adjust the font size (recommended: 60–100px for social media)
4. Click **Display Text** to render it big
5. Use **Full Screen** for big-screen viewing
6. Click **Copy Text** to copy and paste anywhere
7. Toggle theme or clear to start over
